Class: Bomber

Bomber(scene, x, y, key, textureName, characterId, healthPoints, movementSpeed, uid)

Bomber Class. One character type of the player. Holds the same properties as a regular player but have the ability to use bomb bullet.

Constructor

new Bomber(scene, x, y, key, textureName, characterId, healthPoints, movementSpeed, uid)

sets up the bomber object. calls createWeapon and createSpecialWeapon adds the sprite to the scene

Parameters:

The name of the texture that is used for the bomber

healthPoints number 500

The health that a bomber will have in the game
movementSpeed number 64

the speed that the bomber moves
