Class: Player

Player(scene, x, y, key, textureName, characterId, healthPoints, movementSpeed, uid)

Player Class. The actual player sprite that gets added the the scene which is controlled by the user or bot or player data from the database

Constructor

new Player(scene, x, y, key, textureName, characterId, healthPoints, movementSpeed, uid)

sets up the player object. calls createWeapon and createSpecialWeapon adds the sprite to the scene
