Class: Units

Units(scene, x, y, barx, bary, name, type, healthPoints, speed, range, cooldown, uid)

Units Class. The class that defines and applies the properties of the towers that created. Including tower attacking function.

Constructor

new Units(scene, x, y, barx, bary, name, type, healthPoints, speed, range, cooldown, uid)

Init the properties for units objects. Currently it is mainly about towers. Create tower attacking function, add tower sprites and their hp bars to scene.

towerAttack(tower, target, time)

Function to let the tower shoot correct target if the target is in its attack range. The attack of the tower will have a cooldown that is assigned to it. When the attack range,cooldown are fine, the tower is ready to attack the target. The target is the one that is nearest to the tower which is found from the findnearenemy function inside towerAttack.
