Class: emptyBar

emptyBar(scene, x, y)

Class that generate a empty backing status bar on the screen, thus we can make the status bars in the game to have the cutting or reducing effect by cropping the front bar.

Constructor

new emptyBar(scene, x, y)

Generate and display an empty bar on the game screen with giving x and y value

Parameters:
Name Type Description
scene phaser.Scene

the phaser scene which the instance of this class will appears on
x Number

the x position value on the game screen
y Number

the y position value on the game screen
