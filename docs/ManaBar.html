Class: ManaBar

ManaBar(scene, x, y, type, value, uid)

Class that generates and displays a front blue bar which will be representing the mana point of the character

Constructor

new ManaBar(scene, x, y, type, value, uid)

Generates and displays the front mana bar on the screen with giving x and y value, and assign health point value and unique ID for it
